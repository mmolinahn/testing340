Running the Server:
After unzipping the folder, go into it in the command line and enter the following command. (IF USING UNIX (MAC OR LINUX), REPLACE ';'s with ':'s)

java -cp server.jar;shared.jar;gson-2.6.2.jar;json.jar communication.ServerCommunicator

Emulator to Use:
Nexus 9 API 28 (Tablet)
